# My Node Static Server starter projects

### Includes
* Node
* Express
* Bower
  * jQuery
  * Ember
  * Handlebars
* Gulp
* Stylus
* Coffeescript

### Setup

1. Clone this repo
2. run `npm install`


### Running

1. For asset generation: `gulp watch`
2. Run the server: `npm start`
3. Browse to: `http://localhost:3002`
